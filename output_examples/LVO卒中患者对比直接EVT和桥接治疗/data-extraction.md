## 1. 数据提取方法说明

### 1.1 双人独立提取原则

所有数据提取工作遵循**双人独立提取**原则，由两名具备神经内科/循证医学背景的研究人员独立进行。具体流程如下：

- **提取工具：** 使用预先设计的标准化数据提取表格（基于本文档第 2–6 节所列字段），在 Microsoft Excel 或 Covidence 平台中执行。
- **提取对象：** 已发表的完整期刊论文（Full-text article）及官方补充材料（Supplementary Materials/Appendix），以原始发表文献为第一依据。
- **提取范围：** 涵盖 protocol.md 第8.2节所规定的全部六个模块变量。
- **记录方式：** 每名提取者独立填写相同格式的数据提取表，提取完成后由第三人汇总比对，不得在提取阶段相互参阅对方结果。

### 1.2 预测试（Pilot Test）

正式提取开始前，两名提取者共同对同一篇纳入 RCT（建议选取 DIRECT-MT，因其样本量最大、数据最完整）进行预提取，对比结果，识别以下问题：

- 字段定义的歧义性（如 sICH 定义标准的差异：SITS-MOST vs. ECASS III）
- 数据格式转换规则（如从百分比反算事件例数）
- 数据缺失时的记录规范（统一使用"NR"标注未报告项，"NC"标注需计算项）

预测试完成后召开共识会议，修订提取指南细则，方可开始正式提取。

### 1.3 分歧解决机制

两名提取者完成独立提取后，执行以下分歧解决流程：

```
步骤1：系统比对两名提取者的结果，逐字段标注一致/分歧
步骤2：针对分歧字段，两名提取者共同回阅原文对应段落，讨论协商
步骤3：协商后达成一致 --> 记录最终值及解决说明
步骤4：协商后仍无法达成一致（如原文描述模糊、数据来源有争议）
       --> 提交第三名高级研究人员（仲裁者）独立审阅，裁决结果即为最终值
步骤5：在提取记录中注明分歧类型（"文献歧义"/"计算差异"/"理解分歧"）
       及解决方式（"协商一致"/"仲裁裁决"）
```

### 1.4 数据核查与转换规则

提取过程中执行以下标准化数据核查与转换：

| 情形 | 处理规则 |
|------|---------|
| 百分比转换为例数 | 事件数 n = 百分比 × 总人数 N，结果四舍五入至整数；原始百分比保留以供核查 |
| SE 转换为 SD | SD = SE × √N |
| Median/IQR 转换为 Mean/SD | 采用 Wan et al. (2014) 方法（BMC Med Res Methodol）；在缺失数据说明中标注 |
| CI 转换为 SE | SE = (UCL - LCL) / (2 × 1.96)（正态近似）；OR 等对数尺度效应量先取对数再转换 |
| 数据仅以图形呈现 | 使用 WebPlotDigitizer（https://automeris.io/WebPlotDigitizer）提取，在备注中注明"图形提取" |
