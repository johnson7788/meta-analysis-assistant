### 5.2 PRISMA 2020 官方 R 包数据输入参数

以下为使用 `PRISMA2020` R 包时所需输入的标准化参数，可直接复制到 R 脚本中：

```r
# 安装并载入 PRISMA2020 包
# install.packages("PRISMA2020")
library(PRISMA2020)

# 填入本综述的 PRISMA 数据
data <- PRISMA_data(
  # 来源鉴定（Identification）
  database_results   = 670,   # 数据库检索合计（不含其他来源）
  other_results      = 15,    # 其他来源（手工检索、灰色文献等）

  # 去重（Deduplication）
  duplicates         = 265,   # 去重删除合计

  # 标题/摘要筛选（Title/Abstract Screening）
  records_screened   = 420,   # 去重后进入筛选
  records_excluded   = 385,   # 初筛排除

  # 全文获取（Retrieval）
  dbr_sought_reports = 35,    # 申请获取全文
  dbr_notretrieved   = 1,     # 无法获取全文

  # 全文评估（Eligibility）
  dbr_assessed       = 34,    # 全文评估
  dbr_excluded       = 28,    # 全文排除

  # 纳入（Included）
  new_studies        = 6,     # 纳入系统综述
  new_quant          = 6      # 纳入 Meta 分析
)

# 生成流程图
PRISMA_flowdiagram(data,
  detail_databases = TRUE,
  # 可添加各数据库细节
  previous_studies = 0,
  previous_reports = 0
)
```
