## 1. 纳入排除标准概要（Eligibility Criteria Summary）

本节摘自 protocol.md 第5节，用于指导所有筛选决策。

### 1.1 PICOS 框架

| PICOS 要素 | 定义 |
|-----------|------|
| **P（人群）** | 急性大血管闭塞（LVO）缺血性卒中成年患者（年龄 ≥18岁），经 CTA/MRA/DSA 证实 ICA 末段、MCA M1/M2 段、基底动脉或椎动脉颅内段急性闭塞；发病前 mRS ≤2分；在 EVT 治疗时间窗内 |
| **I（干预）** | 直接血管内治疗（Direct EVT）：机械取栓或其他血管内再通治疗，不预先给予任何静脉溶栓药物 |
| **C（对照）** | 桥接治疗（Bridging Therapy）：静脉注射 alteplase（0.9 mg/kg）或 tenecteplase（0.25 或 0.4 mg/kg）后进行 EVT；对照组必须为计划性桥接治疗（均预定接受 EVT） |
| **O（结局）** | 主要：90天 mRS 0–2 功能独立率；次要：90天全因死亡率、sICH 发生率、成功再通率（mTICI 2b–3）|
| **S（研究设计）** | 仅纳入 RCT（优效性或非劣效性设计均可），需发表于同行评审期刊的完整报告 |

### 1.2 纳入标准（Inclusion Criteria）

| 代码 | 标准 |
|------|------|
| **I1** | 原始随机对照试验（RCT），含优效性、非劣效性或等效性设计；允许单中心或多中心 RCT |
| **I2** | 成年患者（年龄 ≥18岁）；经影像学证实急性 LVO 缺血性卒中；发病前 mRS ≤2分；在 EVT 治疗时间窗内 |
| **I3** | 干预组：直接 EVT（不予前置静脉溶栓）；对照组：桥接治疗（静脉溶栓 [alteplase 或 tenecteplase] + EVT）；两组均须进行 EVT |
| **I4** | 至少报告以下结局之一：90天 mRS 0–2、90天全因死亡率、sICH 发生率、成功再通率（mTICI 2b–3 或等效标准）|
| **I5** | 已发表于同行评审期刊的完整研究报告（Full-text article）|

### 1.3 排除标准（Exclusion Criteria）

排除标准按以下优先顺序应用（遇第一个适用条目即停止，记录该条目）：

| 代码 | 标准 | 说明 |
|------|------|------|
| **E1** | 研究设计不符 | 观察性研究（队列/病例对照/横断面）、病例系列/报告、系统综述/Meta 分析、仅会议摘要形式（无对应完整期刊论文）、动物研究、综述性文章 |
| **E2** | 研究人群不符 | 儿科人群（年龄 <18岁）、非 LVO 缺血性卒中（小血管病变、腔隙性梗死等）、出血性卒中、以 TIA 为主要研究对象 |
| **E3** | 干预/对照不符 | 非直接 EVT vs. 桥接治疗的比较（如单纯 IVT vs. EVT）、对照组不进行 EVT、干预组在 EVT 前使用了溶栓药物、两组治疗策略均无 EVT |
| **E4** | 数据不可获取 | 无法从已发表文献、补充材料或联系作者后获取任何预定义结局数据；数据仅以图形呈现且联系作者无回应 |
| **E5** | 重复发表 | 同一临床试验数据的重复报告（保留最完整/最新版本，其余排除并注明） |

## 2. 筛选方法与流程概述

### 2.1 双人独立筛选原则

所有筛选工作均遵循**双人独立筛选**原则：

- **标题/摘要筛选阶段（初筛）：** 两名研究人员（筛选者A、筛选者B）各自独立阅读所有去重后文献的标题和摘要，独立作出纳入/排除/不确定的决策，期间不相互沟通。
- **全文筛选阶段（复筛）：** 经初筛进入全文阶段的文献由两名研究人员各自独立获取并阅读全文，依据纳入/排除标准作出最终判断，并记录排除原因代码。
- 任何一名研究人员判断为"纳入"或"不确定"的文献即进入下一阶段（保守原则，疑问时纳入）。仅当两名研究人员均独立判断为"排除"时，该文献方可排除。
- 分歧通过两人讨论协商解决；如仍无法达成一致，由**第三名高级研究人员（仲裁者）**独立审阅并作出最终裁决。

### 2.2 推荐筛选工具

| 工具 | 网址 | 推荐理由 |
|------|------|----------|
| **Covidence**（首选） | https://www.covidence.org/ | Cochrane 官方合作平台，支持双人筛选、冲突标注、自动统计 kappa；适合正式系统综述 |
| **Rayyan**（备选） | https://www.rayyan.ai/ | 免费、支持 AI 辅助标注、支持团队协作；适合资源有限的研究团队 |
| DistillerSR | https://www.distillersr.com/ | 企业级功能，支持高度定制化工作流 |
| ASReview | https://asreview.nl/ | 开源，AI 主动学习辅助筛选，适合大规模初筛 |

**导入格式：** 将文献管理软件（Zotero/Endnote）中去重后的文献导出为 RIS 格式，导入上述筛选平台。

### 2.3 一致性标准

- 正式筛选开始前，两名筛选人员共同对**同一批预选文献（50篇）**进行试筛（Pilot Screening），计算初步 Cohen's kappa。
- **目标 kappa ≥ 0.80**（"almost perfect agreement"）。若 kappa 未达标，须召开共识会议，进一步澄清纳入/排除标准中的歧义条目，重新试筛，直至 kappa 达标后方可开始正式筛选。
- 正式筛选全程分阶段记录 kappa 值，最终在报告中呈现标题/摘要和全文两个阶段的 kappa 值及对应百分比一致率。

### 2.4 分歧解决机制

```
步骤1：两名筛选者完成独立筛选，系统记录分歧条目
步骤2：两名筛选者进行面对面（或视频）共识讨论，逐条审议分歧
步骤3：若共识讨论达成一致，记录最终决策及解决方式（"协商达成一致"）
步骤4：若共识讨论仍无法达成一致，提交第三名仲裁者独立审阅
步骤5：仲裁者决策作为最终决策，记录仲裁结果
步骤6：在筛选者间一致性报告中，按"协商前"和"最终（协商/仲裁后）"分别报告
```

## 3. 标题摘要筛选表（Title/Abstract Screening Form）

### 3.1 筛选问题清单

基于纳入/排除标准，初筛阶段使用以下5个核心筛选问题：

| 问题编号 | 筛选问题 | 判断依据 |
|---------|---------|---------|
| **Q1** | 研究对象是否为急性缺血性卒中患者，且明确提及或提示存在大血管闭塞（LVO）？ | 标题/摘要中应出现"large vessel occlusion"/"LVO"/"acute ischemic stroke"/"proximal occlusion"/"ICA occlusion"/"MCA occlusion"/"basilar artery occlusion"等词，或明确描述接受了机械取栓的缺血性卒中患者 |
| **Q2** | 研究是否同时涉及血管内治疗（EVT/机械取栓）？ | 出现"endovascular treatment"/"mechanical thrombectomy"/"thrombectomy"/"EVT"等词 |
| **Q3** | 研究是否同时涉及静脉溶栓治疗（alteplase/tenecteplase）或桥接治疗的比较？ | 出现"intravenous thrombolysis"/"alteplase"/"tenecteplase"/"bridging therapy"/"bridging treatment"/"tPA"/"rt-PA"等词 |
| **Q4** | 研究设计是否为随机对照试验（RCT）？ | 出现"randomized"/"randomised"/"RCT"/"non-inferiority"/"randomized controlled trial"等词；明确排除综述、Meta 分析、观察性研究、病例报告 |
| **Q5** | 研究是否为成人患者（年龄 ≥18岁）？ | 无儿科字样（"pediatric"/"neonatal"/"infant"/"children"），默认成人群体时判断为"是" |

### 3.2 决策规则

```
初筛决策规则（Title/Abstract Stage Decision Rules）：

规则1（纳入）：Q1=是 AND Q2=是 AND Q3=是 AND Q4=是 AND Q5=是
               --> 决策：纳入（Include）- 进入全文筛选

规则2（不确定）：以下任一情形 --> 决策：不确定（Uncertain）- 进入全文筛选
  (a) 任一问题因信息不足无法判断
  (b) 摘要未完整报告方法学信息
  (c) Q1/Q2/Q3/Q4 判断为"可能是"但证据不充分
  (d) 仅有会议摘要，需获取全文确认是否有对应期刊论文

规则3（排除）：以下任一情形 --> 决策：排除（Exclude）
  (a) Q4=否（明确非 RCT：综述、Meta 分析、观察性研究、病例报告/系列）
  (b) Q1=否（明确非 LVO 缺血性卒中，如非缺血性卒中、TIA、脑出血等）
  (c) Q2=否（明确无 EVT/机械取栓）
  (d) Q3=否（明确无溶栓治疗成分，无法涉及直接 EVT vs. 桥接的比较）
  (e) Q5=否（儿科/新生儿人群）

原则：当不确定是否排除时，倾向于纳入（当存在疑虑时，进入全文）。
```

## 4. 全文筛选表（Full-text Screening Form）

### 4.1 逐条纳入标准核查清单

| 标准代码 | 核查项目 | 判断 |
|---------|---------|------|
| I1 | 研究设计确认为 RCT（优效性/非劣效性/等效性均可）？随机分配方法是否在文中描述？ | 是 / 否 |
| I2 | 研究人群为成年（≥18岁）急性 LVO 缺血性卒中患者，经影像学确认？发病前 mRS ≤2 或等效基线功能状态？在 EVT 时间窗内？ | 是 / 否 |
| I3 | 干预组为直接 EVT（明确不予前置溶栓）？对照组为桥接治疗（静脉 alteplase 或 tenecteplase + EVT）？两组均接受 EVT？ | 是 / 否 |
| I4 | 至少报告以下一项：90天 mRS 0–2 / 90天全因死亡率 / sICH 发生率 / mTICI 2b–3 再通率？ | 是 / 否 |
| I5 | 已发表于同行评审期刊的完整研究报告（非会议摘要、非预印本）？ | 是 / 否 |

### 4.2 排除原因代码（分层优先顺序）

全文排除时，按以下优先顺序逐条检查，遇第一个满足的条目即记录该代码并停止：

| 排除代码 | 排除原因 | 具体判断标准 |
|---------|---------|------------|
| **E1** | 研究设计不符 | 确认为观察性研究（队列/病例对照/横断面）、病例系列/报告、系统综述/Meta 分析、仅会议摘要无完整期刊论文版本、叙述性综述、注册研究方案（无结果数据） |
| **E2** | 研究人群不符 | 儿科/新生儿（年龄 <18岁）；非 LVO 卒中（小血管病变、腔隙性梗死、静脉窦血栓等）；出血性卒中；以 TIA 为主要研究对象；仅限于模拟/规划研究 |
| **E3** | 干预/对照不符 | 比较为单纯 IVT vs. EVT（非"直接 EVT vs. 桥接"格局）；对照组不进行 EVT（即仅接受 IVT）；干预组在 EVT 前接受了溶栓药物；两组比较的干预措施均不涉及 EVT；比较不同取栓装置或技术而非溶栓策略 |
| **E4** | 数据不可获取 | 全文和补充材料均未报告任何预定义结局（mRS 0–2/死亡率/sICH/mTICI）；数据仅以图形呈现且已联系通讯作者（等待 ≥4周）仍无回应 |
| **E5** | 重复发表 | 确认为同一 RCT 的重复报告或子集分析（如亚组结果单独发表而主要结局数据与已纳入研究完全重叠）；如为补充数据可合并提取则不排除 |

### 7.4 Cohen's kappa 计算说明

Cohen's kappa 按以下公式计算：

```
κ = (Po - Pe) / (1 - Pe)

其中：
Po = 观测一致率（Observed proportion of agreement）
   = (一致判断纳入数 + 一致判断排除数) / 总文献数

Pe = 期望一致率（Expected proportion of agreement by chance）
   = [(筛选者A判纳入数/N) × (筛选者B判纳入数/N)] +
     [(筛选者A判排除数/N) × (筛选者B判排除数/N)]

N = 总文献数
```

**计算说明：** "不确定"在计算时按偏保守原则处理——将其归类为"倾向纳入"类别，以确保 kappa 计算时不高估一致性。最终计算和报告将使用 R 软件（irr 包或 vcd 包的 Kappa 函数）执行。
